# Schedule API

A RESTful API for managing schedules

## Features

- Create, read, update, and delete schedules

## Requirements

- PHP 8.2 or higher
- Composer
- Symfony
- Docker

## Installation

Install dependencies:

```bash
composer install
```

Configure your environment:

- Copy `.env.example` to `.env`
- Update database configuration in `.env`
